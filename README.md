<h1># 📊 Employee Dashboard - React Internship (Day 6)</h1>



<h3>## ✨ Features</h3>
* **🔒 Authentication System:** Secure Login and Logout functionality.
* **💾 Persistent Sessions:** Uses Local Storage to keep the user logged in even after a page refresh.
* **🌍 Global State Management:** Context API is used to manage the authentication state across the entire application without prop drilling.
* **📡 API Integration:** * **GET Request:** Fetches real-time fake employee data using Axios.
    * **POST Request:** Allows the admin to add new employees to the list dynamically.
* **👤 Admin Profile:** Displays the currently logged-in administrator's details.
* **💻 Responsive & Clean UI:** Designed with professional, modern styling.

## 🛠️ Tech Stack
* **Frontend:** React.js (Vite)
* **State Management:** Context API
* **Storage:** Local Storage
* **HTTP Client:** Axios
* **Styling:** CSS-in-JS (Inline Modern Styles)
