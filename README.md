<h1># 📊 Employee Dashboard - React Internship (Day 6)</h1>



<h3>## ✨ Features</h3>
🔒 Authentication System:Secure Login and Logout functionality.
<br>
💾 Persistent Sessions:Uses Local Storage to keep the user logged in even after a page refresh.<br>
🌍 Global State Management:Context API is used to manage the authentication state across the entire application without prop drilling.<br>
📡 API Integration:<b>GET Request:Fetches real-time fake employee data using Axios.</b><br>
POST Request:Allows the admin to add new employees to the list dynamically.<br>
👤 Admin Profile:** Displays the currently logged-in administrator's details.<br>

💻 Responsive & Clean UI: Designed with professional, modern styling.<br>

## 🛠️ Tech Stack
* Frontend:React.js (Vite)
* State Management:Context API
* Storage: Local Storage
* HTTP Client:Axios
*Styling:CSS-in-JS (Inline Modern Styles)
